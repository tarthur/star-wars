# Star-wars guide

This project was bootstrapped with Create React App.

You can find the most recent version of this app [here](http://starwars-visualguide.com).

## About

This site is a copy of the site [starwars-visualguide.com](https://starwars-visualguide.com/#/"). The purpose of the site is to show my ability to reuse code while working with higher-order components, render functions, etc.

The api of this site was provided by [swapi.co](https://swapi.co/)


## Technology stack

  * `React` `Router`

  * `SCSS` `React CSS modules`
  * `Bootstrap`
